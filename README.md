# Лабораторная работа №6
**Тема: Решение задач регрессии с помощью нейронных сетей.**

## Задачи

### 1. Прогнозирование данных HeadHunter (Lite)
Разработали 6 различных архитектур нейронных сетей для работы с табличными данными сервиса HeadHunter.
- **Особенности данных:** использование исключительно числовых параметров из подготовленной таблицы, исключая подачу текстовых признаков.
- **Методы:** эксперименты с гиперпараметрами слоев для поиска модели с минимальной ошибкой предсказания.
- **Результат:** проведен сравнительный анализ архитектур и выбран оптимальный вариант для оценки соискателей.

### 2. Прогноз исходов баскетбольных матчей (Pro)
Реализовали нейронную сеть для предсказания результатов спортивных событий на основе исторической базы матчей.
- **Задача:** оптимизировать архитектуру сети для достижения средней абсолютной ошибки (**MAE**) не более **17 очков**.
- **Инструменты:** настройка глубокой нейронной сети для выявления скрытых закономерностей в статистике команд.
- **Результат:** достигнута требуемая точность прогноза, позволяющая оценивать результативность матчей с минимальной погрешностью.

### 3. Предсказание стоимости автомобилей с ЮЛЫ (Ultra Pro)
Создали комплексную модель для оценки рыночной стоимости автомобилей на основе данных площадки ЮЛА.
- **Алгоритм:** разделение данных на обучающую, тестовую и проверочную выборки; использование среднеквадратичной ошибки (**MSE**) в качестве функции потерь.
- **Архитектура:** выходной слой настроен на 1 нейрон (линейная регрессия); внедрена процедура денормировки предсказанных значений для возврата к реальному диапазону цен.
- **Результат:** реализован расчет суммарного процента ошибки (отношение среднего модуля ошибки к средней цене). Модель демонстрирует высокую адекватность при оценке стоимости транспортных средств.
